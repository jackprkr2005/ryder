# Ryder — brand assets

Ryder is the social home for golf societies. The mark is a brass flagstick
and ball; the wordmark is set in Fraunces.

## Files
| File | Use |
|------|-----|
| `ryder-lockup.svg` / `.png` | Primary logo — mark + wordmark + tagline (light backgrounds) |
| `ryder-wordmark.svg` / `.png` | Wordmark + tagline only |
| `ryder-icon.svg` | App icon / favicon source (brass mark on green tile) |
| `icon-32/180/192/512.png` | Rendered app-icon / favicon sizes |
| `ryder-mark-brass.svg` | Mark only, brass — for dark / photographic backgrounds |
| `ryder-mark-green.svg` | Mark only, green — single-colour on light backgrounds |
| `ryder-mark-on-green.png` | Brass mark previewed on the brand green |

## Colours
| Token | Hex | Use |
|-------|-----|-----|
| Green | `#16432f` | Primary brand / wordmark |
| Green 2 | `#1d5a3e` | Gradient top / hovers |
| Brass / Gold | `#a9823a` | The mark, accents |
| Ink | `#1a241d` | Body text |
| Ink soft | `#646d63` | Secondary text / tagline |
| Paper | `#fffdf8` | Surfaces |
| Background | `#f4f0e7` | App background |

Icon tile gradient: `#1d5a3e → #143f2d → #0f2e22` (top → bottom).

## Type
- **Fraunces** — display & wordmark (weight 600).
- **Manrope** — UI text & tagline (the tagline is uppercase, ~0.16em tracking).

## Usage
- Keep clear space around the logo equal to the height of the flag's ball.
- Don't recolour the mark outside the palette, rotate it, or stretch it.
- On busy/dark photos use `ryder-mark-brass.svg`; on light single-colour
  contexts use `ryder-mark-green.svg`.
- The SVG wordmark uses live Fraunces text — outline it to paths in your design
  tool if a recipient may not have the font installed.
